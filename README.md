# 🏥 MediSphere AI - Hospital Management System

Built by Zarak Khan

## 🌟 What is MediSphere AI?
A complete AI-powered hospital management system built with Flask and Grok API.

## ✨ Features
- 🤖 AI Chat Assistant (Multi-Agent System)
- 👤 Patient Registration
- 📅 Appointment Booking
- 🧪 Lab Report Checking
- 🔢 Live Queue Status
- 🚨 Emergency Contact Info
- 📊 Admin Dashboard

## 🛠️ Technologies Used
- Python + Flask
- Grok API (xAI)
- SQLite Database
- HTML, CSS, JavaScript

## ⚙️ Setup
1. Clone this repo
2. Install requirements: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your Grok API key
4. Run: `python app.py`
5. Open: `http://localhost:5000`
